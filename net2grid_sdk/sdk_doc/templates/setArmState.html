SDK / net2gridApiModule / iasDevicesService /

setArmState(code, mode)

Changes the global arming state of all the IAS-sensors in the ZigBee network.

Parameters

Name Type Details
code string Password code of the intruder alarm system.
mode string Global arming state mode.

Example code

myAppModule.controller('myAppCtrl', function(iasDevicesService) {

	iasDevicesService.setArmState('12341234', 'disarm');
});
