createRadioFrequencyPlugPayload(plug, systemCode, unitCode, value, group)

The createRadioFrequencyPlugPayload(plug, systemCode, unitCode, value, group) function is used to create the payload of the radioFrequencyPlug(plug, systemCode, unitCode, value, group) function.

Parameters

Name Type Details
plug Object The virtual SmartPlug object.
systemCode number
 The system code determines groups of manufacturer specific SmartPlugs.
unitCode number
 The unit code labels individual manufacturer specific SmartPlugs.
value number
 The state of the SmartPlug (0/1)
group boolean
 If true, all SmartPlugs with the same system code will receive the command.

Returns

Name Type Details
payload Array
number		 The payload sent to the NET2GRID SmartBridge.
