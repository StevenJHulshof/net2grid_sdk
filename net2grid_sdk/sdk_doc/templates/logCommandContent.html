logCommandContent(command)

The logCommandContent(command) function is basically the same as the logAttributeContent(attribute) function, only this one prints the command data into the console. This function can also be used as a development tool.

Parameters

Name Type Details
command Object The command object

Example code

myAppModule.controller('myAppCtrl', function(zigbeeLutService, generalDevicesService) {

	var toggleCommand = zigbeeLutService.getToggleCommand();
	
	generalDevicesService.logCommandContent(toggleCommand);
});
