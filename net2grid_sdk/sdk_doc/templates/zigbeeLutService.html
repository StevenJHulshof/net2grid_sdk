zigbeeLutService

This AngularJS service contains the ZigBee cluster library, and can be seen as the first component of the NET2GRID SDK.

Returns

Name Type Details
getClusterLut() Function Returns cluster look up table.
getLogClusterLut() Function Returns log cluster look up table.
getIasLut() Function Returns intruder alarm systems look up table.
getArmStateLut() Function Returns arming status look up table.
getDataTypeLut() Function Returns data type look up table.
getStatusLut() Function Returns status look up table.
getArgDataTypeLut() Function Returns argument data type look up table.
get{{attribute.nameFun}}Attribute Function Returns {{attribute.name}} attribute.
get{{command.nameFun}}Command Function Returns {{command.name}} command.

myAppModule.controller('myAppCtrl', function(zigbeeLutService) {

	//Code here to access the zigbeeLutService
