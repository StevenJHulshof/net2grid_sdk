logAvailableCommands()

This method does the same as the logAvailableAttributes() method, but prints the common available commands of the clusters present in the virtual device. This method can also be used as a development tool.

Example code

myAppModule.controller('myAppCtrl', function(ipAddressService, generalDevicesService) {

	ipAddressService.setIp('123.456.7.89');

	generalDevicesService.setupGeneralDevices();
	
	$scope.generalDevices;
	
	$scope.$on('generalDevices.update', function(event) {
		
		$scope.generalDevices = generalDevicesService.getGeneralDevices();
	});

	$scope.logAvailableCommands = function() {
	
		$scope.generalDevices[0].logAvailableCommands();
	}
});
