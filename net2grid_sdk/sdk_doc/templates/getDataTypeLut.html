getDataTypeLut()

The getDataTypeLut() function returns the dataTypeLut object.

Returns

Name Type Details
dataTypeLut Array
Object		 Shows length of used data types in octets, name, id, and if the data type is 'analog', 'discrete' or none of these.

Example code

myAppModule.controller('myAppCtrl', function($scope, zigbeeLutService) {

	$scope.dataTypeLut = zigbeeLutService.getDataTypeLut();
});
