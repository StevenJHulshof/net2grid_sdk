checkReportableChangeNeeded(attributes, reportableChange)

The checkReportableChangeNeeded(attributes, reportableChange) function checks whether the attributes need a reportable change parameter. When the attribute data type is 'analog', a reportable change parameter is needed.

Parameters

Name Type Details
attributes Array
Object		 The array containing the attribute objects.
reportableChange Array
number		 The array containing the reportable change values.

Returns

Name Type Details
check boolean Returns true when the attributes and the reportable value input matches.

Example code

myAppModule.controller('myAppCtrl', function(utilityService, zigbeeLutService) {

	var onOffAttribute = zigbeeLutService.getOnOffAttribute();
	
	if(utilityService.checkReportableChangeNeeded([onOffAttribute], [null])){
	
		//Do something
	}
});
