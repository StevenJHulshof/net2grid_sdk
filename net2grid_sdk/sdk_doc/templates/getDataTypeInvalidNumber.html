getDataTypeInvalidNumber(id)

The getDataTypeInvalidNumber(id) function returns the invalid number of a data type based on its identification code.

Parameters

Name Type Details
id number The numeric identification code of the data type.

Returns

Name Type Details
invalidNumber number The invalid number of the data type.

Example code

myAppModule.controller('myAppCtrl', function(utilityService) {

	var invalidNum = utilityService.getDataTypeInvalidNumber(0x22); //ID of INT24U
});
