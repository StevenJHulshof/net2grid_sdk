sensorDevices

The sensorDevices object contains a list of all IAS-sensors found in the ZigBee network. Each IAS-sensor has the following methods available.

Methods

Name Type Details
getMask() Function Returns the arming state mask of an individual IAS-sensor.
setMask(code, mask) Function Changes the arming state mask of an individual IAS-sensor.
