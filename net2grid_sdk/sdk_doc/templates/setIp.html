setIp(ip)

The setIp(ip) function is used to store the IP-address of the NET2GRID SmartBridge into the ipAddressService. This function must be called prior to using the generalDevicesService, iasDevicesService and the radioFrequencyDevicesService.

Parameters

Name Type Details
ip string New IP address.

Example code

myAppModule.controller('myAppCtrl', function(ipAddressService) {

	ipAddressService.setIp('123.456.7.89');
});
