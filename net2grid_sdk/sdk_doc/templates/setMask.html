setMask(code, mask)

Changes the arming state mask of an individual IAS-sensor.

If the call is successful, the mask settings in the IAS-sensor will be changed. To update the virtual devices, run the setupIasSensors() function again.

Parameters

Name Type Details
code string Password code of the intruder alarm system.
mask string The arming mask name string

Example code

myAppModule.controller('myAppCtrl', function($scope, ipAddressService, iasDevicesService) {

	ipAddressService.setIp('123.456.7.89');

	iasDevicesService.setupIasSensors();
	
	$scope.sensorDevices;
	$scope.mask;

	$scope.$on('sensorDevices.update', function(event) {
	
		$scope.sensorDevices = iasDevicesService.getSensorDevices();
	});
	
	$scope.setMask = function() {
	
		$scope.sensorDevices[0].setMask('12341234', 'bypass');
	}
});
