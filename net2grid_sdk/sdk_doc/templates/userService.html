userService [WAN only]

The userService is used for storing user data, the SmartBridge EUI and to request the access token from the server. The userService can be injected into any other service, controller or directive.

Returns

Name Type Details
setUserData(clientId, clientSecret, userName, passWord)() Function Stores user data into service.
getUserData() Function Returns user data.
getToken() Function Returns token from service.
postTokenRequest() Function Post token request to server.
setEui(euiStr) Function Stores EUI into service.
getEui() Function Returns EUI.

Example code

myAppModule.controller('myAppCtrl', function(userService) {

	//Code here to access the userService
});
