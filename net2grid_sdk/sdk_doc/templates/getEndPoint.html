getEndPoint(clusterId, endPoints)

The getEndPoint(clusterId, endPoints) function is used to determine whether a cluster is available in the specified endpoints. The return value contains both the boolean check and the endpoint which contains the cluster. This function is used in the generalDevicesService to check whether a cluster is available in the virtual device when making a HTTP call.

Parameters

Name Type Details
clusterId number The numeric identification code of a cluster.
endPoints Array The endpoint array available in a virtual device.

Returns

Name Type Details
buf Array An array with an index of 2. The first index contains a boolean check whether the cluster is available in de specified endpoints. The second index is the number of the endpoint which holds the specified cluster.

Example code

myAppModule.controller('myAppCtrl', function($scope, utilityService, generalDevicesService, zigbeeLutService) {

	$scope.$on('generalDevices.update', function(event) {
		
		$scope.generalDevices = generalDevicesService.getGeneralDevices();
	});
	
	var basicCluster = zigbeeLutService.getClusterLut().Basic;
	
	var buf = utilityService.getEndPoint(basicCluster, generalDevices.endPoints);
});
