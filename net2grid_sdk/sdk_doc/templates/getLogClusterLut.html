getLogClusterLut()

The getLogClusterLut() function returns the logClusterLut object.

Returns

Name Type Details
logClusterLut Array
Object		 Contains cluster id, name, attributes and commands of all clusters.

myAppModule.controller('myAppCtrl', function($scope, zigbeeLutService) {

	$scope.logClusterLut = zigbeeLutService.getLogClusterLut();
});
