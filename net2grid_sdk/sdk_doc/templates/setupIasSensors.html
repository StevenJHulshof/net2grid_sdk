setupIasSensors()

This function creates a list of virtual IAS-sensors corresponding to the actual sensors in the ZigBee network.

Make sure to set te IP address before calling the setupIasSensors() function. The function will broadcast a 'sensorDevices.update' event which can be watched before calling the getSensorDevices() function. When failed, an error message will be printed in the browser console, containing potential data and the HTTP status.

Example code

myAppModule.controller('myAppCtrl', function(ipAddressService, iasDevicesService) {

	ipAddressService.setIp('123.456.7.89');

	iasDevicesService.setupIasSensors();
});
