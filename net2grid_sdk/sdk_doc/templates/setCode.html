SDK / net2gridApiModule / iasDevicesService /

setCode(oldCode, newCode)

Changes the passcode of the Intruder Alarm System. This code will be used when sending commands to the IAS-sensors in the ZigBee network.

If the call is successful, the code stored in the iasDevicesService will also be changed. When failed, an error message will be printed in the browser console, containing potential data and the HTTP status.

Parameters

Name Type Details
oldCode string Old password code of the intruder alarm system.
newCode string New password code of the intruder alarm system.

Example code

myAppModule.controller('myAppCtrl', function(iasDevicesService) {

	iasDevicesService.setCode('12341234', '56785678');
});
