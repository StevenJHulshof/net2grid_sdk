logGlobalCommandClusterError(clusterId)

The logGlobalCommandClusterError(clusterId) function is used to print the missing cluster error to the browser console. This function is used in the generalDevicesService for logging the missing cluster error.

Parameters

Name Type Details
clusterId number The numeric identification code of a cluster.

Example code

myAppModule.controller('myAppCtrl', function(utilityService, zigbeeLutService) {

	var basicCluster = zigbeeLutService.getClusterLut().Basic;
	
	utilityService.logGlobalCommandClusterError(basicCluster);	
});
