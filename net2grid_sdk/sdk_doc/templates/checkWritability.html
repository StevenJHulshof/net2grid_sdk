checkWritability(attributes)

The checkWritability(attributes) function can be used to check whether attributes are writable. This function is used in the writeAttributes(clusterId, attributes, value) method.

Parameters

Name Type Details
attributes Array
Object		 Array of attribute objects.

Returns

Name Type Details
writability Array
number
boolean		 When returning false, the second index of the array contains the attribute identification code.

Example code

myAppModule.controller('myAppCtrl', function(zigbeeLutService, utilityService) {

	var onOffAttribute = zigbeeLutService.getOnOffAttribute();
	var zclVersionAttribute = zigbeeLutService.getZclVersionAttribute();
	
	var writability = utilityService.checkWritability([onOffAttribute, zclVersionAttribute]);
});
