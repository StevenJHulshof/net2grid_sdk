checkAttrAvailability(clusterId, attributes)

The checkAttrAvailability(clusterId, attributes) function determines whether the specified attributes are available in the specified cluster.

Parameters

Name Type Details
clusterId number The numeric identification code of a cluster.
attributes Array
Object		 The array containing the attribute objects.

Returns

Name Type Details
check boolean Returns true when the attributes are present in the specified cluster.

Example Code

myAppModule.controller('myAppCtrl', function($log, utilityService, zigbeeLutService) {

	var basicCluster = zigbeeLutService.getClusterLut().Basic;
	var zclVersionAttribute = zigbeeLutService.getZclVersionAttribute();
	var manufacturerNameAttribute = zigbeeLutService.getManufacturerNameAttribute();
	
	if(utilityService.checkAttrAvailability(basicCluster, [zclVersionAttribute, manufacturerNameAttribute])) {
	
		// Do something
	}
	else {
	
		$log.error("Chosen Cluster ID and attribute input do not match!");
	}
});
