setEui(euiStr)

The setEui(euiStr) function stores the EUI of the used SmartBridge into the userService.

Parameters

Name Type Details
euiStr string Hexadecimal EUI string.

Example code

myAppModule.controller('myAppCtrl', function(userService) {

	userService.setEui("84df0c0000000004");
});
