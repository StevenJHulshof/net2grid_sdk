checkCluster(clusterId)

This method will check whether the input cluster parameter is available in the virtual device. Useful for checking the functionality of the device.

Parameters

Name Type Details
clusterId number The numeric identification code of a cluster.

Returns

Name Type Details
check boolean Returns true if the cluster is available within the device.

Example code

myAppModule.controller('myAppCtrl', function(ipAddressService, generalDevicesService, zigbeeLutService) {

	ipAddressService.setIp('123.456.7.89');

	generalDevicesService.setupGeneralDevices();
	
	$scope.generalDevices;
	
	$scope.$on('generalDevices.update', function(event) {
		
		$scope.generalDevices = generalDevicesService.getGeneralDevices();
	});

	$scope.checkCluster = function() {
	
		var onOffCluster = zigbeeLutService.getCLusterLut().OnOff;
		
		if($scope.generalDevices[0].checkCluster(onOffCluster)) {
		
			//Do something
		}
	}
});
