setupGeneralDevices()

The setupGeneralDevices() function will scan the ZigBee network for devices and creates virtual duplicates which the app developer can use to control the actual devices.

Make sure to set te IP address before calling the setupGeneralDevices() function. The function will broadcast a 'generalDevices.update' event which can be watched before calling the getGeneralDevices() function. When failed, an error message will be printed in the browser console, containing potential data and the HTTP status.

Example code

myAppModule.controller('myAppCtrl', function(ipAddressService, generalDevicesService) {

	ipAddressService.setIp('123.456.7.89');

	generalDevicesService.setupGeneralDevices();
});
