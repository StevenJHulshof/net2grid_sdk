utilityService

The utilityService contains some utility purpose functions. Most of these functions are used for building the other servivces available in the net2gridApiModule. Check the detailed documentation of the functions itself to see their use in mobile applications. The utilityService can be injected into any other service, controller or directive.

Returns

Name Type Details
euiToHex(euiDec) Function Returns hexadecimal value of eui.
getEndPoint(clusterId, endPoints) Function Checks whether cluster is available and returns end point.
checkAttrAvailability(clusterId, attributes) Function Checks whether the chosen cluster contains the chosen attributes.
logGlobalCommandClusterError(clusterId) Function Prints missing cluster error message to console.
getDataTypeLength(id) Function Checks datatype length of payload id.
convertPayload(payload, length) Function Converts response payload to correct value (ReadAttributes).
argArrayCheck(command, args) Function Checks whether the argument matches with the array tag.
checkWritability(attributes) Function Check whether attributes are writable.
createWriteAttributesPayload(attributes, value) Function Create payload for the writeAttributes command.
getStatus(value) Function Get status from look up table.
getDataTypeName(id) Function Returns data type name.
createReadReportingConfigurationPayload(attributes, directions) Function Create payload of the readReportingConfiguration command.
getDataTypeAdFromId(id) Function Checks whether datatype is 'analog' or 'discrete' using id.
parsePayloadNum(payload) Function Parses payload to one value.
getDataTypeInvalidNumber(id) Function Returns invalid number of chosen data type.
createConfigureReportingSendPayload(attributes, timeouts) Function Create payload of ConfigureReportingSend command.
getDirectionString(num) Function Returns direction as a string.
checkReportableChangeNeeded(attributes, reportableChange) Function Checks whether the reportableChange argument is needed.
getDataTypeAdFromName(name) Function Checks whether datatype is 'analog' or 'discrete' using name.
getDataTypeIdFromName(name) Function Returns id of chosen datatype name.
createConfigureReportingReceivePayload(attributes, minInterval, maxInterval, reportableChange) Function Creates payload of ConfigureReportingReceive command.
createRadioFrequencyPlugPayload(plug, systemCode, unitCode, value, group) Function Creates payload of radioFrequencyPlug command.
createWriteCommandPayload(args, command) Function Creates payload of writeCommand command.
checkArmState(armState, mask) Function Returns active arm state.
convertMaskType(mask) Function Converts mask string into number.
createReadAttributesPayload(attributes) Function Creates payload of readAttributes command.
