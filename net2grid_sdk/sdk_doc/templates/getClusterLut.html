getClusterLut()

The getClusterLut() function returns the clusterLut object.

Returns

Name Type Details
clusterLut Object Contains all clusters with their cluster id.

Example code

myAppModule.controller('myAppCtrl', function($scope, zigbeeLutService) {

	$scope.clusterLut = zigbeeLutService.getClusterLut();
	$scope.basicCluster = $scope.clusterLut.Basic;
});
