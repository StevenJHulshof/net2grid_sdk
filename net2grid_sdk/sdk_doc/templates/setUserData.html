SDK / net2gridApiModule / userService /

setUserData(clientId, clientSecret, userName, passWord)

The setUserData(clientId, clientSecret, userName, passWord) function is used to store the user data into the userService.

Parameters

Name Type Details
clientId string Identification of the client developer.
clientSecret string Authorization of the client developer.
userName string User name
passWord string password

Example code

myAppModule.controller('myAppCtrl', function(userService) {

	userService.setUserData('demoapp', 'demopass', 'zll', 'hK86u8G7');
});
