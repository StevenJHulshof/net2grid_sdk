radioFrequencyDevicesService

The radio frequency devices are taken care of by the radioFrequencyDevicesService. This service doesn't make use of the ZigBee specifications, for it works with radio frequencies. The radioFrequencyDevicesService contains a few preconfigured SmartPlug objects which can be used to control the corresponding manufacturer specific SmartPlugs. There is also the option to add a custom SmartPlug, if the SmartPlug you are using is not available in the radioFrequencyDevicesService.

Injections

Name Type Details
$log Service Simple Angular service for logging.
ipAddressService Service Service handling the IP address.
utilityService Service Service handling all utility functions.

Returns

Name Type Details
customPlug(name, bitPeriod, protocol, systemCodeBitLength, unitCodeBitLength) Constructor Custom smart plug.
radioFrequencyPlug() Function Controls manufacturer specific smartplugs.
getImpulsPlug() Function Returns Impuls smart plug data.
getBrennenstuhlRCS1000NPlug() Function Returns Brennenstuhl RCS-1000N smart plug data.
getClickOnClickOffAPA22300RPlug() Function Returns ClickOnClickOff APA2-2300R smart plug data.
getIntertechnoITR1500Plug() Function Returns Intertechno ITR-1500 smart plug data.
getElroHomeEasyHE877Plug() Function Returns Elro HomeEasy HE877 smart plug data.
getSiemensRC018Plug() Function Returns Siemens RC018 smart plug data.
