customPlug(name, bitPeriod, protocol, systemCodeBitLength, unitCodeBitLength)

The customPlug(name, bitPeriod, protocol, systemCodeBitLength, unitCodeBitLength) object constructor can be used to add custom manufacturer specific SmartPlugs to your application.

Parameters

Name Type Details
name string The name of the custom plug.
bitPeriod number The period of the bits sent to the RF plug.
protocol number The number of the protocol used.
systemCodeBitLength number The system code bit length determines the maximum value for this parameter.
unitCodeBitLength number The unit code bit length determines the maximum value for this parameter.

Example code

myAppModule.controller('myAppCtrl', function($scope, radioFrequencyDevicesService) {

	$scope.myCustomPlug = new radioFrequencyDevicesService.customPlug('My custom plug', 260, 3, 5, 5');
});
