parsePayloadNum(payload)

The parsePayloadNum(payload) function parses the input array to a numeric value.

Parameters

Name Type Details
payload Array
number		 The payload array response.

Returns

Name Type Details
data number A single numeric value.

Example code

myAppModule.controller('myAppCtrl', function(utilityService) {

	var data = utilityService.parsePayloadNum([1, 0, 123]);
});
