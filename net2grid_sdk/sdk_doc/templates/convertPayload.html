convertPayload(payload, length)

The convertPayload(payload, length) function is used to convert the payload of the readAttributes(clusterId, attributes) function to usable values. This function is only used in the readAttributes(clusterId, attributes) function.

Parameters

Name Type Details
payload Array
number		 The payload array.
length string
number		 The length of the payload data type, or the string representation of the data type name.

Returns

Name Type Details
value string
number		 Usable converted value of the payload.
