getStatus(value)

The getStatus(value) function returns the string value of a numeric status response identification.

Parameters

Name Type Details
value number The numeric identification code of a status, present in response commands.

Returns

Name Type Details
status string The string value of the status.

Example code

myAppModule.controller('myAppCtrl', function(utilityService) {

	var statusStr = utilityService.getStatus(0x00); // Status 'SUCCESS' 
});
