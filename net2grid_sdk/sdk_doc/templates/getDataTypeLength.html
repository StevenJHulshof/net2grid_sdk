getDataTypeLength(id)

The getDataTypeLength(id) functions is used to get the length of a data type based on its identification code.

Parameters

Name Type Details
id number The numeric identification code of a data type.

Returns

Name Type Details
octets number The length of the data types is returned in octets.

Example code

myAppModule.controller('myAppCtrl', function(utilityService) {

	var octets = utilityService.getDataTypeLength(0x22); //ID of INT24U
}]);
