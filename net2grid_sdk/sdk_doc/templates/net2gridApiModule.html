net2gridApiModule

The net2gridApiModule can be injected into your main application module. Doing this gives you acces to all the services available in the net2gridApiModule.

Content

Name Type Details
userService [WAN only] Service Service handling the user data, SmartBridge EUI and access token.
ipAddressService [LAN only] Service Service handling the IP address.
utilityService Service Service handling all utility functions.
generalDevicesService Service Service handling all general devices.
iasDevicesService Service Service handling Intruder Alarm System (IAS) devices.
radioFrequencyDevicesService Service Service handling radio frequency plugs.
zigbeeLutService Service Service handling all look up tables used in the Net2Grid SDK.

Example code

var myAppModule = angular.module('myAppModule', [

	'net2gridApiModule'
]);
