getIp()

The getIp() function returns the stored IP-address of the NET2GRID SmartBridge. The IP-address must first be set before it can be returned to the user.

Returns

Name Type Details
ipZgd string The IP address of the used ZGD.

Example code

myAppModule.controller('myAppCtrl', function($scope, ipAddressService) {

	$scope.ipZgd = ipAddressService.getIp();
});
