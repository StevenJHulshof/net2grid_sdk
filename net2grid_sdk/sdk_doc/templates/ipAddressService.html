ipAddressService [LAN only]

The ipAddressService is used to store the IP-address of the NET2GRID SmartBridge. Initializing the IP-address is needed before running the setup functions in the generalDevicesService and the iasDevicesService. The ipAddressService can be injected into any other service, controller or directive.

Returns

Name Type Details
getIp() Function Returns stored IP address.
setIp(ip) Function Changes the stored IP address, used to address the ZGD.

Example code

myAppModule.controller('myAppCtrl', function(ipAddressService) {

	//Code here to access the ipAddressService
});
