getArmState()

Returns the global arming state of all the IAS-sensors in the ZigBee network. Make sure to set the global arming state before calling this function.

Returns

Name Type Details
armState string The global arming state

Example code

myAppModule.controller('myAppCtrl', function($scope, iasDevicesService) {

	iasDevicesService.setArmState('12341234', 'day');
	
	$scope.globalArmState = iasDevicesService.getArmState();
});
