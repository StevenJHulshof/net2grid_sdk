iasDevicesService

The IAS library is used for setting up and controlling IAS-sensors. Because the IAS library uses different calls in the NET2GRID REST API, the IAS library has been seperated from the general devices library. For security reasons, a passcode must be sent when making calls to IAS-sensors in the ZigBee network. The iasDevicesService allows for modifying the passcode, changing the global arming state of the sensors and the arming state of the individual sensors. This service can scan the ZigBee network for IAS-sensors.
