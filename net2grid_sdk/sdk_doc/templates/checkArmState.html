checkArmState(armState, mask)

The checkArmState(armState, mask) function returns the active arming status of an individual IAS-sensor.

Parameters

Name Type Details
armState string Global arming state
mask number The arming mask

Returns

Name Type Details
activeArmState number The active arming status.

Example code

myAppModule.controller('myAppCtrl', function(utilityService) {

	var activeArmState = utilityService.checkArmState('day', 1);
});
