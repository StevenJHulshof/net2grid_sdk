getDataTypeIdFromName(name)

The getDataTypeIdFromName(name) function returns the data type identification code based on the data type name.

Parameters

Name Type Details
name string The name string of a data type.

Returns

Name Type Details
id number The numeric identification code of a data type.

Example code

myAppModule.controller('myAppCtrl', function(utilityService) {

	var dataTypeId = utilityService.getDataTypeIdFromName('INT24U'); 
});
