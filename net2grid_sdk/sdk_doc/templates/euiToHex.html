euiToHex(euiDec)

The euiToHex(euiDec) function converts the numeric decimal value of an EUI into a hexadecmal representation as a string. This function can be used to display the numeric EUI of a virtual device as the actual hexadecimal value of the EUI.

Parameters

Name Type Details
euiDec Array
number		 Decimal value of the EUI as a number.

Returns

Name Type Details
euiHex Array
string		 Hexadecimal value of the EUI as a string.

Example code

myAppModule.controller('myAppCtrl', function($scope, utilityService) {

	$scope.euiHexStr = utilityService.euiToHex([0,13,111,0,0,208,227,251]);
});
