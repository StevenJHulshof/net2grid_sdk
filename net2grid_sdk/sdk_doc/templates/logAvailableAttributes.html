logAvailableAttributes()

The developer can call this method to print a list in the browser console of the common available attributes of the clusters present in the virtual device. To specify the attribute set of the device, use the discoverAttributes(clusterId) method. The logAvailableAttributes() method can be used as a development tool.

Example code

myAppModule.controller('myAppCtrl', function(ipAddressService, generalDevicesService) {

	ipAddressService.setIp('123.456.7.89');

	generalDevicesService.setupGeneralDevices();
	
	$scope.generalDevices;
	
	$scope.$on('generalDevices.update', function(event) {
		
		$scope.generalDevices = generalDevicesService.getGeneralDevices();
	});

	$scope.logAvailableAttributes = function() {
	
		$scope.generalDevices[0].logAvailableAttributes();
	}
});
