getGeneralDevices()

The getGeneralDevices() function return the list of general devices when called. It is recommended to watch the 'generalDevices.update' event to make sure all general devices have been created before calling them.

Returns

Name Type Details
generalDevices Array
Object		 Contains all general devices as an array.

Example code

myAppModule.controller('myAppCtrl', function(ipAddressService, generalDevicesService) {

	ipAddressService.setIp('123.456.7.89');

	generalDevicesService.setupGeneralDevices();
	
	$scope.generalDevices;
	
	$scope.$on('generalDevices.update', function(event) {
		
		$scope.generalDevices = generalDevicesService.getGeneralDevices();
	});
});
