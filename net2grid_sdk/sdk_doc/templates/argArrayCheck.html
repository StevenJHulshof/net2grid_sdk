argArrayCheck(command, args)

The argArrayCheck(command, args) function checks whether an argument matches with the 'array' tag in the command object.

Parameters

Name Type Details
command Object The cluster specific command object
args Array The array of arguments. Arrays can be nested inside this array.

Returns

Name Type Details
arrCheck boolean Returns true if the argument matches with the 'array' tag of the cluster specific command.

Example code

myAppModule.controller('myAppCtrl', function(zigbeeLutService, utilityService) {

	var moveToColorCommand = zigbeeLutService.getMoveToColorCommand();
	
	if(utilityService.argArrayCheck(moveToColorCommand, [0x4000, 0x9900, 20])) {
	
		//Do something
	}
});
