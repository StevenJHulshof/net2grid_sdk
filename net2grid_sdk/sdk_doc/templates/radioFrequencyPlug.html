SDK / net2gridApiModule / radioFrequencyDevicesService /

radioFrequencyPlug(plug, systemCode, unitCode, value, group)

This function changes the state of a RF-plug. If the function fails to deliver the promise, an error message is printed into the console. Catching the promise is not a necessity, but it is recommended to do so, for it contains information about the used RF-pluf and the status of the ZGD request.

Parameters

Name Type Details
plug Object The RF plug object.
systemCode number The system code of the RF-plug.
unitCode number The unit code of the RF-plug.
value number The value sent to the RF-plug (0/1)
group boolean Determines whether all RF-plugs with the same system codes are triggered.

Returns

Name Type Details
promise Object Name of the plug, value of its state, status.

Example code

myAppModule.controller('myAppCtrl', function(radioFrequencyDevicesService) {

	var impulsPlus = radioFrequencyDevicesService.getImpulsPlug();
	
	radioFrequencyDevicesService.radioFrequencyPlug(impulsPlug, 0, 0, 1, false).then(function(promise) {
	
		var promise = promise;
	}); // Set Impuls smartplug to ON
});
