convertMaskType(mask)

The convertMaskType(mask) function returns the numeric value of the mask based on the mask name string.

Parameters

Name Type Details
mask string The arming mask name string

Returns

Name Type Details
maskNum number The numeric value of the arming mask.

Example code

myAppModule.controller('myAppCtrl', function(utilityService) {

	var maskNum = utilityService.convertMaskType('night');
});
